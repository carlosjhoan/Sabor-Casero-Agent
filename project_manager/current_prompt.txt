## 📋 ESTÁNDARES DE CÓDIGO

### Arquitectura:
Estándares de código para el proyecto Sabor Casero Assistant.
Se sigue Clean Architecture con separación en Domain, Application e Infrastructure.
Comentarios en español, nombres de variables y funciones en inglés.


### Capas:
- Domain: Capa de dominio - contiene las reglas de negocio puras.
NO debe tener dependencias externas (ni de infrastructure, ni de application).

- Application: Capa de aplicación - orquesta los casos de uso.
Depende de domain, pero NO de infrastructure (por interfaz).

- Infrastructure: Capa de infraestructura - implementaciones concretas.
Depende de domain y application (implementa interfaces).


### Convenciones de nombres:
- Clases: PascalCase
- Funciones: snake_case
- Variables: snake_case

### Docstrings: Google style en Español
