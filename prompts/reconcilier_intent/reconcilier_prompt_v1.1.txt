## DECISION LOGIC FLOW:

###  STEP 1: CLASSIFY REQUEST TYPE
Analyze the request for these patterns:

A. QUANTITY REDISTRIBUTION (Example 1):

Pattern: "Uno de los X con modificaciones" or "X de los Y con Z"
Logic: Find matching item(s), adjust quantities, create new item with modifications
Operation: SPLIT_AND_MODIFY

B. DISH REPLACEMENT (Example 2):

Pattern: "Cambie X por Y" or "Mejor Y"
Logic: Replace dish entirely (preserve quantity unless specified)
Operation: REPLACE_DISH

C. COMPONENT MODIFICATION (Example 3):

Pattern: "en vez de A, B" or "cambie A por B"
Logic: Change specific component (protein, side, size) of existing dish
Operation: MODIFY_COMPONENT

D. PROTEIN SWAP (Example 4):

Pattern: "con pollo frito" or "mejor con res"
Logic: Change protein across matching dishes
Operation: UPDATE_PROTEIN_GLOBAL

###  STEP 2: MAP TO SPECIFIC OPERATIONS
OPERATION DEFINITIONS:

####  SPLIT_AND_MODIFY:

Use when: User wants "one of X with modifications" from existing items
Action: Reduce quantity of source item, create new item with modifications
Example: "Uno de las carnes asadas sin arroz" → split 1 from carne_asada, create new with "sin arroz"

####  REPLACE_DISH:

Use when: User wants to change entire dish
Action: Remove old dish, add new dish (preserve quantity if possible)

Example: "Cambie bandeja paisa por bocachico frito" → replace dish entirely

####  MODIFY_COMPONENT:

Use when: User wants to change specific component (side, protein, size)
Action: Update specific field(s) on existing item

Example: "en vez de fríjol, con verdura" → change side_dish from "fríjol" to "verdura"

####  UPDATE_QUANTITY:

Use when: Only quantity changes

Action: Adjust quantity up or down

####  ADD_MODIFIER:

Use when: Adding/removing specific ingredients
Action: Append to special_instructions ("sin X", "extra Y")

####  UPDATE_PROTEIN_GLOBAL:

Use when: Changing protein across multiple similar items
Action: Update protein_choice for all matching items

###  STEP 3: HANDLE COLOMBIAN RESTAURANT SPECIFICS

####  COMMON PATTERNS TO RECOGNIZE:

"más bien" = correction, usually means REPLACE or MODIFY
"regáleme con" = polite request for protein/side change
"será que me" = polite request for modification
"sin X, con Y" = remove X, add Y
"en vez de" = replacement of component

####  COMPONENT MAPPINGS:

Protein choices: "pollo", "res", "cerdo", "pescado", "bocachico", "mixto", "vegetariano"
Sides: "arroz", "fríjol", "lentejas", "verdura", "papa", "yuca", "ensalada"
Sizes: "corriente", "mini"
Modifiers: "frito", "asado", "a la plancha", "guisado"
